# selenium_quickstart

## Refer
http://www.kaizenprogrammer.com/entry/2016/11/23/143938

## Precondition
* PC : mac
* Browser : google chrome

## Procedure

### Add chromedriver

```
wget https://chromedriver.storage.googleapis.com/2.25/chromedriver_mac64.zip
unzip chromedriver_mac64.zip
```

### Install node package

```
npm install
```


### Execute mocha

```
node quickstart.js
```
